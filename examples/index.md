# Example skills

Five complete, working skills you can copy into your own setup today. Each example is a real, drop-in `.md` file — not a snippet to adapt. If you save one to your `.claude/skills/` folder, Claude will start firing it on relevant requests immediately.

| Skill | What it drafts |
|---|---|
| [FBA Quickstart](fba-quickstart) | NYSED Part 200-aligned Functional Behavior Assessments |
| [ABC Analyzer](abc-analyzer) | Function hypothesis from raw ABC data |
| [IEP Behavioral Goal](iep-behavioral-goal) | Measurable annual goal + objectives |
| [Parent Update](parent-update) | 1-2 week plain-language family update |
| [Teacher Coaching Note](teacher-coaching-note) | Post-observation fidelity + coaching point |

{: .tip }
> See [Lesson 5 — Installing skills](../lessons/05-installing-skills) for where to save these files so Claude picks them up automatically.
