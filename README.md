# Claude Cowork Skills for School-Based ABA Consultation

A teaching module for BCBAs, school psychologists, and behavior consultants who want to use Claude Cowork (Claude Code) to streamline their school-based practice — without learning to code.

> **You'll learn how to write Markdown (`.md`) files that turn Claude into a domain-aware assistant for FBAs, BIPs, IEP goals, parent communication, and teacher coaching.**

---

## Who this is for

You if you:
- Practice school-based ABA in any capacity (BCBA, BCaBA, school psychologist, behavior specialist, special ed coordinator).
- Already use Claude Cowork or Claude Code, even casually.
- Find yourself re-typing the same instructions to Claude every consultation cycle ("write this in plain language for parents," "format like a NYSED-compliant FBA," etc.).
- Want Claude to *automatically* bring the right framework, vocabulary, and structure to recurring consultation tasks.

You don't need any programming background. If you can edit a Word document, you can write a skill.

---

## What you'll be able to do by the end

1. Explain what a Claude skill is and when one beats just typing a prompt.
2. Read and modify the frontmatter and body of a `.md` skill file.
3. Write a brand-new skill from scratch for a recurring task in your own caseload.
4. Install your skills into Claude Cowork so they trigger automatically on the right requests.
5. Share skills with colleagues so a whole team consults more consistently.

---

## How to work through this module

Read the lessons in order. Each one is short — 5–10 minutes. After lesson 3 you should be able to write a usable first skill.

| # | Lesson | What it covers |
|---|---|---|
| 01 | [What is a skill?](lessons/01-what-are-skills.md) | The concept, how it differs from a regular prompt, why it matters for repetitive consultation work |
| 02 | [Anatomy of a skill file](lessons/02-anatomy-of-a-skill.md) | Frontmatter, body, the `name` / `description` fields that make Claude pick the right skill |
| 03 | [Writing skills that actually fire](lessons/03-writing-effective-skills.md) | Trigger language, what to include, what to leave out, common mistakes |
| 04 | [ABA-specific patterns and recipes](lessons/04-aba-recipes.md) | Templates for FBAs, BIPs, IEP goals, parent updates, teacher coaching, treatment integrity |
| 05 | [Installing and sharing skills](lessons/05-installing-skills.md) | Where files live, how to test, how to share with a team |

The [`examples/`](examples/) folder contains five complete, working skills you can copy into your own setup today.
