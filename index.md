## Who this is for

You if you:

- Practice school-based ABA in any capacity (BCBA, BCaBA, school psychologist, behavior specialist, special ed coordinator).
- Already use Claude Cowork or Claude Code, even casually.
- Find yourself re-typing the same instructions to Claude every consultation cycle ("write this in plain language for parents," "format like a NYSED-compliant FBA").
- Want Claude to *automatically* bring the right framework, vocabulary, and structure to recurring consultation tasks.

You don't need any programming background. **If you can edit a Word document, you can write a skill.**

## What you'll be able to do by the end

1. Explain what a Claude skill is and when one beats just typing a prompt.
2. Read and modify the frontmatter and body of a `.md` skill file.
3. Write a brand-new skill from scratch for a recurring task in your own caseload.
4. Install your skills into Claude Cowork so they trigger automatically on the right requests.
5. Share skills with colleagues so a whole team consults more consistently.

## How to work through this module

Read the [lessons](lessons/) in order. Each is 5–10 minutes. After Lesson 3 you should be able to write a usable first skill.

The [examples](examples/) folder contains five complete, working skills — drop them straight into your Claude Cowork setup and they'll start firing on relevant requests today.

## A note from the author
